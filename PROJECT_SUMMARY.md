# Project Summary

The AI Commerce Intelligence Platform is an end-to-end decision and execution system for early-stage commerce businesses.

The platform predicts the best:

- micro niche
- selling/growth platform
- dense audience slice

using three separate machine learning models.

These model outputs are combined through a final decision engine, which generates a business recommendation and interprets the business stage and product style.

On top of that, an AI execution agent converts the final recommendation into actionable execution tasks across niche, platform, audience, and growth strategy.

The final system is delivered through a Streamlit dashboard where a user can select a business, generate the report, view recommendations, and download the result.

Separately, the project also demonstrates simulated upstream data-engineering capability through a Shopify API demo, a Square API demo, and a canonical merged dataset that shows how source systems could be standardized in a production environment.
